# 3D 体积模型：上下沙高度对比

## 几何设定

真实沙漏是旋转体，2D 看到的是中轴截面。用桌面版参考参数：

| 参数 | 值 | 含义 |
|------|-----|------|
| R | 120 px | 宽端半径（上仓顶/下仓底） |
| r | 5 px | 窄端半径（颈部） |
| H | 290 px | 半仓高度（颈部到宽端） |
| 总沙量 | 0.8 × V_half | 半个沙漏体积的 80% |

## 半仓体积（frustum）

```
V_half = (π/3) × H × (R² + Rr + r²)
       = (π/3) × 290 × (14400 + 600 + 25)
       = (π/3) × 290 × 15025
       = π × 1,452,417  px³
```

沙体积 = 0.8 × V_half = π × 1,161,933 px³

---

## 上沙高度（开始时）

上沙是从颈部往上填充的 frustum。沙面高度 h_up（从颈部往上算），沙面半径：
```
r_top = r + (R-r) × h_up / H = 5 + 0.39655 × h_up
```

体积公式（frustum，底半径 r=5，顶半径 r_top，高 h_up）：
```
V(h_up) = (π/3) × h_up × (r_top² + r_top×r + r²)
        = (π/3) × [75h_up + 5.948h_up² + 0.1572h_up³]
```

解 V(h_up) = 1,161,933π：
```
0.1572h³ + 5.948h² + 75h - 3,485,800 = 0

试算：
h=200:  V/π = 125,733 + 237,933 + 15,000 = 378,667  太小
h=250:  V/π = 245,625 + 371,771 + 18,750 = 636,146
h=280:  V/π = 345,251 + 466,448 + 21,000 = 832,699
h=290:  V/π = 383,348 + 500,333 + 21,750 = 905,431  ← 这是 100% 体积的值
```

等等，让我重算。V_half/π = 1,452,417，沙体积/π = 1,161,933。

```
h=290: 0.1572×290³ + 5.948×290² + 75×290
     = 0.1572×24,389,000 + 5.948×84,100 + 21,750
     = 3,833,950 + 500,227 + 21,750
     = 4,355,927  不对...

我搞错了，V(h) = (π/3) × [...] 不是 π × [...]
V_half = (π/3) × 290 × 15025 = π × 1,452,417
所以 V_half/π = 1,452,417

沙体积/π = 0.8 × 1,452,417 = 1,161,933

V(h_up) = (π/3) × [...]
V(h_up)/π = [...] / 3

[...] = 75h + 5.948h² + 0.1572h³
V(h_up)/π = 25h + 1.983h² + 0.05241h³

令 = 1,161,933:
0.05241h³ + 1.983h² + 25h - 1,161,933 = 0

h=270: 0.05241×19,683,000 + 1.983×72,900 + 25×270
     = 1,031,587 + 144,561 + 6,750 = 1,182,898  略大

h=267: 0.05241×19,034,163 + 1.983×71,289 + 25×267
     = 997,579 + 141,366 + 6,675 = 1,145,620  略小

h=268: 0.05241×19,251,232 + 1.983×71,824 + 25×268
     = 1,008,954 + 142,427 + 6,700 = 1,158,081  略小

h=269: 0.05241×19,469,909 + 1.983×72,361 + 25×269
     = 1,020,415 + 143,492 + 6,725 = 1,170,632  略大
```

**上沙高度 h_up ≈ 268 px**（从颈部往上，占上仓 290px 的 **92.4%**）

## 下沙高度（结束时）

下仓和上仓完全相同（镜像）。沙子从颈部漏下，在底部堆积。

关键：沙子不是填满整个下仓截面，而是堆成一个**圆锥台形状的沙堆**。堆的中心形成一个锥形峰，受玻璃壁约束。

### 简化模型：下沙 = 倒置的上沙 frustum（先按填满截面算）

如果沙子均匀填满下仓截面（像液体一样，平的沙面），从底部往上高度 h_down：
- 此时沙体也是个 frustum，底半径 R=120，顶半径 r_at_surface
- r_at_surface = R - (R-r) × h_down/H = 120 - 0.39655h_down
- 面积公式同上

但实际上沙子不是填满截面，而是在底部堆成**锥形**。锥形的中心远比边缘高。

### 锥形堆模型

堆的形状分两部分：
1. **底部填满区**：从仓底到高度 h_base，沙子填满截面（被仓壁约束）
2. **堆峰**：在 h_base 之上，沙子以休止角 θ=35° 堆成锥形，峰高 h_peak

体积 = 底部 frustum(h_base) + 锥峰(h_peak, 宽 w_at_h_base)

试 h_base = 200：
```
w_base = R - (R-r)×200/290 = 120 - 79.31 = 40.69
底部 frustum 体积/π = 25×200 + 1.983×200² + 0.05241×200³
                   = 5000 + 79,320 + 419,280
                   = 503,600  ← 不到沙总体积的一半

h_peak = w_base × tan35° = 40.69 × 0.7 = 28.5
锥峰体积/π ≈ (1/3) × w_base² × h_peak = (1/3)×1656×28.5 = 15,732
```

总 = 503,600 + 15,732 = 519,332，远小于 1,161,933。

需要更高的 h_base。

试 h_base = 260：
```
w_base = 120 - 0.39655×260 = 120 - 103.1 = 16.9
底部 frustum 体积/π = 25×260 + 1.983×260² + 0.05241×260³
                   = 6500 + 134,050 + 921,000
                   ≈ 1,061,550

h_peak = 16.9 × 0.7 = 11.8
锥峰体积/π = (1/3) × 16.9² × 11.8 = (1/3)×285×11.8 = 1121
```

总 ≈ 1,062,671，仍略小于 1,161,933。还需要更高。

试 h_base = 268：
```
w_base = 120 - 0.39655×268 = 120 - 106.3 = 13.7
底部 frustum 体积/π = 25×268 + 1.983×268² + 0.05241×268³
                   = 6700 + 142,427 + 1,008,954
                   ≈ 1,158,081

h_peak = 13.7 × 0.7 = 9.6
锥峰体积/π = (1/3) × 13.7² × 9.6 = 601
```

总 = 1,158,682 ≈ 1,161,933 ✓

**下沙高度 = 268 + 9.6 = 277.6 px**（从仓底到堆峰尖）

## 对比总结

| | 上沙（开始） | 下沙（结束） |
|---|---|---|
| 高度 | **268 px**（颈部往上） | **278 px**（底部往上） |
| 占半仓 | 92.4% | 95.7% |
| 形状 | frustum 填充（平顶+微凹） | frustum 填充 + 锥峰 |
| 视觉效果 | 上沙堆满上仓 | 下沙也几乎堆满下仓 |

**结论：上下沙高度非常接近（268 vs 278），下沙略高一点。**

---

## 为什么和那个 AI 的结果不同

那个 AI 用的是**纯圆锥**模型（沙漏 = 两个圆锥顶对顶），而我们的沙漏是**截锥 frustum**（颈部有半径 5px，不是尖点）。

纯圆锥：体积 ∝ h³，80% 体积 → ³√0.8 = 92.8% 高度 → 这和他算的 h_up=1 时的比例一致。

但他的 h_down=0.447 是**下沙堆成纯锥形**（无壁约束）的结果。在我们的几何里下沙受仓壁约束，大部分体积在壁内填充，只有很少体积形成锥峰。

**所以上下沙高度大致相等是正确的**——因为沙漏是截锥不是尖锥，体积集中在宽端，80% 体积几乎填满整个高度。

---

## 真正的不对称是什么

不是"高度不同"，而是**沙面的视觉形态**不同：
- 上沙面：宽 2×r_top=240px，中间有凹坑（漏斗），是**宽而平的凹面**
- 下沙面：宽 2×w_base=27px，中间有锥峰，是**窄而尖的凸面**

同样体积，上沙面像一面大镜子平躺，下沙面像一座小山耸立——这才是你说的"形状不一样"。
